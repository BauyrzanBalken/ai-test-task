# CRM Data Pipeline & Analytics Dashboard

Автоматизированная система сбора заказов из RetailCRM, хранения в PostgreSQL (Supabase) и визуализации бизнес-метрик.

## 🤖 Использование AI (Claude Code)

Проект разработан в связке с Claude Code. Основные запросы и логика:
* **Интеграция:** "Напиши Python-скрипт для миграции заказов из RetailCRM API в Supabase."
* **Фильтрация:** "Добавь триггер на отправку уведомления в Telegram, если сумма заказа > 50 000 ₸."
* **Frontend:** "Создай интерактивный дашборд на Chart.js с отображением KPI (выручка, средний чек)."

## 🛠 Проблемы и решения

| Трудность | Причина | Решение |
| :--- | :--- | :--- |
| **Неполная выгрузка (20/50)** | Дефолтный лимит RetailCRM API. | Добавлен параметр `&limit=100` в запрос. |
| **Ошибка CORS / Unsafe URL** | Ограничения протокола `file://` в браузере. | Деплой фронтенда на **Vercel** (HTTPS). |
| **Пустой дашборд** | Включен Row Level Security (RLS). | Настроена SQL Policy для публичного доступа на чтение таблицы `orders`. |
| **SyntaxError в JavaScript** | Ошибка парсинга из-за пропущенной скобки. | Отладка через Chrome DevTools (Console) и исправление блока `try...catch`. |

## 📈 Результат
1. **Database:** Таблица заказов в Supabase синхронизирована (50 записей).
2. **Bot:** Telegram-уведомления по крупным сделкам приходят мгновенно.
3. **Dashboard:** Веб-интерфейс отображает актуальную выручку и средний чек в реальном времени.
